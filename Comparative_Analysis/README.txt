# Title of Dataset:
social system, environmental variables, and phylogeny for global avian species. 


## Description of the Data and file structure

This dataset includes two files. One is "global_avian_6553.csv", which includes 6553 avian species with life history and environmental variables.
The other is "global_avian_6553.nex", which is the phylogeny data for global avain species.
